## 3. DB / 마이그레이션 메모

이번 WBS 4에서도 새 마이그레이션을 추가하지 않았다.

근거:
- `groups` 테이블과 `pages.group_id` 컬럼이 이미 존재한다.
- 이번 범위의 핵심 변경은 그룹 CRUD와 이동 정책 구현, 검증 강화, UI 연결이다.

주의:
- 기본 그룹 식별은 현재 `name = '미분류'` 정책에 의존한다.
- 향후 기본 그룹 식별을 더 강하게 고정해야 하면 `is_default` 같은 스키마 확장을 검토할 수 있다.

## 4. 디자인 검토 메모

WBS 7 병행 점검 기준으로 다음 항목을 함께 보정했다.

- 그룹 목록에 이름 변경/삭제 액션 추가
- 에디터 툴바에 현재 노트 그룹 선택 드롭다운 추가
- 그룹 액션 버튼에도 `aria-label` 부여
- 모바일 단일 패널 전환 구조 안에서 그룹 관리 액션 접근 가능 상태 유지
- 그룹 전환 시 목록과 편집기 상태가 어긋나지 않도록 선택 해제 동작 보정

남은 메모:
- 그룹 이름 변경은 현재 `window.prompt` 기반이다.
- 추후 공용 다이얼로그나 인라인 편집 컴포넌트로 승격할지 WBS 5~6과 함께 재검토한다.
