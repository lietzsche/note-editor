## 1. 현재 상태

WBS 5 전환 독립성/충돌은 구현과 핵심 검증까지 완료한 상태다.

완료 항목:
- `dirty / saving / error / conflict` 저장 상태 전이 보강
- 미저장/저장 실패 상태에서 노트 전환, 그룹 전환, 새 노트 생성, 로그아웃, 그룹 이동 가드 추가
- `updated_at` 기반 충돌 감지 및 `force` 강제 저장 경로 추가
- 충돌 비교 다이얼로그와 `덮어쓰기 / 취소` 선택 UI 추가
- 저장 실패 시 로컬 입력 유지 + 재시도 버튼 추가
- 노트 재선택 no-op 처리로 현재 draft 초기화 방지

참고:
- WBS 3 노트 코어와 WBS 4 그룹 관리는 동일 일자 기준 완료 상태를 유지한다.
- 충돌 감지는 현재 `updated_at` 기반 낙관적 잠금으로 동작한다.
