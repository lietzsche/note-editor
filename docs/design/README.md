# Design Docs

이 디렉터리는 제품의 디자인 의사결정과 UI 기준을 관리한다.

## 포맷 정책

1. 기본 포맷은 Markdown(`.md`)으로 사용한다.
2. 디자인 원문(사람이 읽고 수정하는 원본)은 Markdown으로 유지한다.
3. 테마/토큰처럼 기계가 읽어야 하는 데이터는 JSON/TS를 우선 사용한다.
4. XML은 외부 시스템이 XML을 강제할 때만 파생 포맷으로 사용한다.
