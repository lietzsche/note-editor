# Testing Backlog Index

현재 릴리스 범위 밖 기능의 테스트 초안과 수동 검증 시나리오를 관리한다.

## 위치 정책

1. `docs/testing/TEST_PLAN.md`는 현재 릴리스 범위의 공식 테스트 계획만 관리한다.
2. `docs/testing/backlog/*.md`는 backlog 업무용 테스트 초안과 수동 검증 시나리오를 관리한다.
3. backlog 항목이 구현 범위로 승격되면 필요한 내용을 `TEST_PLAN.md`로 올린다.
